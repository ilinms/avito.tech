# Баг-репорты

## BUG-001: Фильтр по статусу "Done" отображает задачи с другими статусами

### Приоритет
**High**

### Компонент
**Список задач (Issues Page)** → Фильтр по статусу

---

## Описание

При применении фильтра по статусу "Done" на странице "Все задачи", система отображает все задачи вместо ожидаемых задач со статусом "Done".

### Шаги для воспроизведения

1. Перейти на страницу "Все задачи" → `https://avito-tech-internship-psi.vercel.app/`
2. Открыть dropdown фильтра "Статус" 
3. Выбрать статус "Done"
4. Проверить список отображаемых задач

### Ожидаемый результат

- Dropdown фильтра должен отобразить выбранный статус "Done"
- Список задач должен содержать **только** задачи со статусом "Done"
- Все видимые chip-элементы статусов должны отображать "Done"

### Фактический результат

- Dropdown фильтра корректно отображает "Done" ✅
- Список задач содержит задачи со статусом "Backlog" и другими ❌
- Фильтрация не применяется к содержимому списка ❌
