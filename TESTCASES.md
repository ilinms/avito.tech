
# TEST CASES

## TC-01 Создание задачи

**Предусловия:**
- Пользователь находится на главной странице сервиса

**Шаги:**
1. Нажать кнопку "Создать задачу"
2. Ввести значение в поле "Название"
3. Ввести значение в поле "Описание"
4. В поле "Проект" выбрать значение "Рефакторинг API"
5. В поле "Приоритет" выбрать значение "Low"
6. В поле "Исполнитель" выбрать значение "Илья Романов"
7. Нажать кнопку "Создать"

**Ожидаемый результат:**
- Созданная задача отображается в списке задач

---

## TC-02 Открытие карточки задачи

**Предусловия:**
- Существует созданная задача
- Пользователь находится на главной странице

**Шаги:**
1. Найти созданную задачу в списке
2. Кликнуть по названию задачи

**Ожидаемый результат:**
- Открывается карточка редактирования задачи
- Отображается заголовок "Редактирование задачи"
